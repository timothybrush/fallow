# SIG audit

1. Read the existing audit and measurement scripts.
2. Run every supported property measurement on the same revision.
3. Record commands, raw evidence locations, limitations, and current scores.
4. Compare with the previous revision using identical settings.
5. Update only claims supported by current measurements.
6. Run repository verification for any changed audit tooling or docs.

Do not estimate missing metrics or turn a proxy into a measured result.
